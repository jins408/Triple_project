# Triple 사전 과제

### 프로젝트 실행 방법
- yarn start test:snapshot

### 사용한 기술
- React
- Javascript
- styled-components

### 선택한 이유
- 보편적이면서 가장 많이 쓰이는 기술이어서 선택했습니다.

### 애니매이션
- state값을 사용해 순서대로 등장하는 애니매이션 구현
- opacity와 transform값을 이용해 살짝 떠오르는 듯한 애니매이션 구현
- setInterval을 사용해서 count 애니매이션 구현
  - 0부터 시작해서 증가값이 점점 줄어들게 구현
